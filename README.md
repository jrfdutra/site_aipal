## Project info
O site da AIPAL Padarias e Pastelarias foi criado para aproximar-nos dos nossos clientes, apresentando de forma clara os nossos horários de funcionamento, contactos e localizações. Nele, é possível conhecer o nosso cardápio, que reflete a tradição da pastelaria portuguesa, com produtos frescos, receitas clássicas e opções pensadas para o dia a dia. O site também conta um pouco da nossa história, marcada pela dedicação, qualidade e paixão pela arte de bem servir, valores que acompanham a AIPAL desde a sua origem e que continuam a guiar o nosso trabalho diariamente.

## What technologies are used for this project?
O Projeto tem as seguintes tecnologias:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
